# Чтобы Claude и ChatGPT стабильно открывались «из Латвии»

Латвия — страна ЕС, и Claude (claude.ai), и ChatGPT в ЕС доступны, поэтому
латвийский выход подходит. Ниже — нюансы, чтобы всё работало без сюрпризов.

## Главное
- С туннелем из этого репозитория трафик к `claude.ai`/`chatgpt.com` идёт через
  Латвию автоматически (это «зарубежные» домены → не попадают в RU-исключение).
- Проверь геолокацию: <https://ipinfo.io> должен показывать Латвию (или ЕС).
- DNS не утекает (FakeIP+DoH), поэтому провайдер не видит обращения к этим
  доменам.

## Если Claude/ChatGPT просит верификацию или «недоступно в вашем регионе»
Почти всегда причина — **старые cookи/кеш** с прежней геолокацией или
несоответствие IP ↔ язык/таймзона. Что сделать:
1. Убедись, что туннель активен и IP латвийский (`ipinfo.io`).
2. Открой сайт в **режиме инкогнито** (свежая сессия без старых cookies).
3. Если не помогло — почисти cookies именно для `claude.ai` / `anthropic.com`
   (Safari: Настройки → Конфиденциальность → Управление данными сайтов).
4. Залогинься заново.

## Стабильность
- Если страница «подвисает» на загрузке — возможно, активный протокол придавили.
  Клиент сам переключится в течение пары минут (`urltest`), либо переключи
  вручную (см. ниже) / перезапусти демон:
  ```bash
  sudo launchctl kickstart -k system/com.user.singbox
  ```
- Для тяжёлых задач (большие ответы, стрим) Hysteria2 (UDP) обычно быстрее, а
  Reality (TCP) — стабильнее и скрытнее. `auto` балансирует по задержке.

## Ручной выбор протокола (если хочется)
Включи Clash API в клиентском конфиге и используй любой Clash-дашборд, либо
просто временно оставь в outbound `proxy.default` нужный тег (`vless-reality`
или `hysteria2`) вместо `auto`, затем:
```bash
bash scripts/install-macos-daemon.sh   # переустановит конфиг
```

## Telegram и YouTube
Тоже идут через туннель (зарубежные адреса). Telegram Desktop/мобильный и
YouTube будут видеть латвийский IP — отдельная настройка не нужна. Если в
Telegram включён собственный встроенный прокси/MTProto — лучше его выключить,
чтобы не было двойного туннелирования.

## Приватность
Весь зарубежный трафик и DNS проходят через твой сервер в Латвии — то есть через
тебя же. Это лучше публичных VPN, но помни: на сервере технически видно
направления соединений. Сервер — твой, держи его в секрете и обновляй.
